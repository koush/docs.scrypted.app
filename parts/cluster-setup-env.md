
Create and edit Scrypted `.env` configuration file:

::: code-group

```sh[Docker Compose]
nano ~/.scrypted/volume/.env
```

```sh[Proxmox]
nano ~/.scrypted/volume/.env
```

```sh[Linux]
nano ~/.scrypted/volume/.env
```

```sh[Mac]
touch ~/.scrypted/volume/.env
open -a TextEdit ~/.scrypted/volume/.env
```

```[Windows Command Prompt]
notepad %USERPROFILE%\.scrypted\volume\.env
```

:::
